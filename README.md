## ⭐ Main Feature
- 이미지 자르기 기능 - canvas API를 활용해 필요한 이미지 부분만 굿즈에 포함
- 관리자 페이지 - 주문조회 기능 구현, 각종 통계 그래프 구현
- 내 정보 페이지 - 구매내역 조회, 주소, 비밀번호 수정, 핸드폰 번호 수정 기능 구현
- 진행 사항 저장 - 중간에 로그인을해도 진행 사항이 저장됩니다.
